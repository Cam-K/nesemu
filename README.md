# Nintendo Entertainment System Emulator
A simple NES emulator, written in C and SDL2

Only compatible with mapper 0 games at the moment.


## How to compile:
``make`` 


## Liberies Needed:
``sudo apt install libcjson-dev libcjson1 libsdl2-dev libsdl2-2.0``

## How to run
### To print help:
``./nesemu -h``

### To run a game:
``./nesemu -n [FILE]


## Controls:
| Keyboard      | NES Contorller   |
|---------------|------------------|
| Arrow Keys    | Control Pad      |
| Z Key         | B Button         |
| X Key         | A Button         |
| Enter         | Start            |
| Shift         | Select           |
